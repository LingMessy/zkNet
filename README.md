## 软件功能

一个校园网账号可以同时认证一台手机和一台电脑（共两台设备）上网，但有的设备会被错误认证（例如PS等游戏机），导致设备冲突无法正常联网。合理的需求无法满足，所以本软件用于修复该错误，解决合法需求。本软件可以将错误认证的设备重新认证为其它设备。如将被错误认证为电脑的设备认证为手机，或将被错误认证为手机的设备认证为电脑。
